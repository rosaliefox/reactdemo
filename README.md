## HTML Template
react的工具包，在head中要引用进来
```html
<!DOCTYPE html>
<html>
  <head>
    <script src="../build/react.js"></script>
    <script src="../build/react-dom.js"></script>
    <script src="../build/browser.min.js"></script>
  </head>
  <body>
    <div id="example"></div>
    <script type="text/babel">

      逻辑代码在这里写

    </script>
  </body>
</html>
```
